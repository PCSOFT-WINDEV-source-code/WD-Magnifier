  
<span style="font-family:Arial sans-serif;font-size:16px;">WD Magnifier</span>

  
<span style="font-family:Arial sans-serif;font-size:14px;">This example enables you to zoom part of the screen with a magnifier.</span>

<span style="font-family:Arial sans-serif;font-size:14px;">The maximum zoom value is set to 8.</span>

<span style="font-family:Arial sans-serif;font-size:14px;">Summary of the example supplied with WINDEV: </span>

<span style="font-family:Arial sans-serif;font-size:14px;">The WLanguge function named dCopyBlt() and the WLanguage property named ..Opacity allow you to perform a capture of the Windows desktop at the location of a window without this window appearing in the screen shot.</span>

<span style="font-family:Arial sans-serif;font-size:14px;">This example uses this tip to zoom the part of the screen found below the magnifier.</span>
